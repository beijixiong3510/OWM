# OWM (Orthogonal Weights Modification)
Code for paper *[Continual Learning of Context-dependent Processing in Neural Networks](https://arxiv.org/abs/1810.01256)*

## Requirements:
- Linux: Ubuntu 16.04

- cuda9.0 & cudnn6.0

- Python 3.5.4

- tensorflow 1.5.0

- tensorflow-gpu 1.4.0

- numpy 1.15.1
